Lynx, utilisé lors du développement des applications chinoise, est un concurrent de taille face à React Native. Il permet d'utiliser n'importe quel bibliothéque frontend, et d'utiliser du CSS natif ! La réalisation des modules natif aux plateformes (android, ios, …) est elle aussi prise en charge de manière simplifiée. Le fait que le compilateur soit en Rust permet d'améliorer le temps de build de la version de développement et de production.
